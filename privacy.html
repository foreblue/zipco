개인정보보호

전용목적

본 확장 프로그램은 사용자가 웹 페이지에서 우편번호를 쉽게 검색하고 입력할 수 있도록 도와주는 도구입니다. 네이버 검색을 통해 우편번호 정보를 가져오고, 자동 입력 기능과 즐겨찾기 기능을 제공하여 사용자의 편의성을 높입니다.

권한별 요청이유

activeTab 권한

  • 현재 활성화된 탭의 웹 페이지에서 우편번호 입력 필드를 자동으로 감지
  • 검색된 우편번호를 해당 입력 필드에 자동으로 입력하기 위해 필요
  • 사용자가 보고 있는 페이지의 DOM 요소에 접근하여 우편번호 입력 필드를 식별

storage 권한

  • 자주 사용하는 주소와 우편번호를 즐겨찾기에 저장
  • 브라우저를 재시작해도 저장된 즐겨찾기 목록 유지
  • 사용자 편의성을 위한 데이터 영구 저장

clipboard-write 권한

  • 검색된 우편번호를 클립보드에 복사하여 다른 곳에 붙여넣기 가능
  • 우편번호 입력의 편의성 향상

https://search.naver.com/* 권한

  • 네이버 검색 결과에서 우편번호 정보를 파싱하여 표시
  • 정확한 우편번호 정보 제공을 위한 네이버 검색 결과 접근

1. 수집하는 정보

  • 우편번호 검색을 위해 입력한 주소 정보
  • 즐겨찾기에 저장한 주소 및 우편번호 정보
  • Chrome Storage API를 통해 저장되는 데이터

※ 위 정보들은 모두 사용자의 브라우저 내부에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 정보 사용 목적

  • 우편번호 검색 및 표시
  • 즐겨찾기 기능 제공
  • 사용자 편의성 향상을 위한 데이터 저장

3. 정보 저장 및 보안

  • 모든 데이터는 사용자의 Chrome 브라우저 내부에만 저장됩니다
  • 서버로 데이터가 전송되지 않습니다
  • Chrome Storage API를 통한 안전한 데이터 저장

4. 제3자 공유

  • 수집된 정보를 제3자와 공유하지 않습니다
  • 외부 서버로 데이터를 전송하지 않습니다

※ 본 확장 프로그램은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

5. 사용자 권리

  • 언제든지 확장 프로그램을 제거하여 저장된 데이터를 삭제할 수 있습니다
  • Chrome 설정에서 저장된 데이터를 직접 삭제할 수 있습니다

6. 데이터 보관 기간

  • 사용자가 확장 프로그램을 제거하거나 데이터를 삭제할 때까지 보관됩니다
  • Chrome Storage API의 제한에 따라 데이터가 저장됩니다

7. 법적 고지사항

  • 본 확장 프로그램은 개인정보보호법을 준수합니다
  • 사용자의 개인정보 보호를 위해 최선을 다하고 있습니다
